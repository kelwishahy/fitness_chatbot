# Fitness Bot (Prototype)
A Facebook Messenger chatbot designed to assist people getting started with weight training

## Built With
* Python 3.7
* Beautiful Soup 4
* Google Dialogflow
* Amazon AWS EC2 Ubuntu 18.04 instance
